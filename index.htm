Links dos projetos:

EXERCÍCIO 01

Crie uma página que contenha:

  • 3 Inputs
  • 1 Botão
  • 1 Div vazia para guardar um resultado
  • Faça uma função Javascript que, ao clicar no botão, some os 3 valores dos inputs e mostre o resultado na Div vazia.
EXERCÍCIO 02

Crie uma página que contenha:

  • 3 Inputs
  • 1 Div vazia para guardar um resultado
  • SEM BOTÃO
  • Faça uma função Javascript que, sempre ao alterar um dos inputs, some os 3 valores dos inputs e mostre o resultado na Div vazia.
EXERCÍCIO 03

Utilizando inputs e botões, realize os desafios abaixo:

  • Crie uma função chamada imprimePares, que não recebe nenhum parâmetro, e que imprime todos números pares entre 1 e 100 no HTML ao clicar em um botão;
  • Crie uma função chamada calculaMedia, que pegue 3 notas de inputs, e que imprima no HTML a média aritmética dos números
  • Crie uma função chamada menorNumero, que leia dois inputs, e que imprime apenas o menor dos números em uma Div de resultado;
  • Crie uma função chamada maiorNumero, que leia dois inputs, e que imprime apenas o maior dos números em uma Div de resultado;
  • (Sem HTML, pode jogar direto no console.log()) Crie uma função chamada inverteArray, que recebe um array de qualquer tipo de dado por parâmetro, e que retorna um array invertido nas posições;
  • (Sem HTML, pode jogar direto no console.log()) Crie uma função chamada encontrarNumero, que recebe um número X por parâmetro e outro parâmetro sendo um array exemplo [2, 7, 8, 3]. A função deve retornar se foi encotrado ou não o número X dentro do array;
